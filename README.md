# Student-Life-Organizer
The student Life Organzier is a web application that allows students to take notes, track finances, and plan their calendar all in one place.

## System Requirements:
### Hardware Requirements:
- Any computer or laptop that can run a modern web browser
- 2 GB of Ram or more
- Basic CPU (no heavy processing)
- Basic peripherals: Keyboard, mouse/trackpad, and a monitor
### Software Requirements:
- Modern Web Browser (Google Chrome, Mozilla Firefox, Microsoft Edge, ect.)
- Must support JavaScript
- Must support localStorage (For saving transactions, notes and events)
## Installation Instructions:
## Teck Stack Overview:
1. Download and set up Visual Studio Code (Html and JavaScript extensions)
2. Download the Live Server extension to run program locally
3. Download or clone the project files (important to maintain same structure)
4. Once project is open in Visual Studio Code, right click "login.html" and select "Open with Live Server"
